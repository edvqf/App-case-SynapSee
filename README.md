# App-case-SynapSee

Para utilizar, é necessário possuir um interpretador python e as seguintes bibliotecas instaladas:
- streamlit
- pandas
- numpy
- joblib
- re
- os
- matplotlib.pyplot

Depois de baixar a pasta "app mental state", clique 2x no arquivo "rodar app".

Abrira um plataforma na web que vai pedir um arquivo tipo planilha com os dados.

Na pasta "app mental state" possui um arquivo de planilha compactada que pode ser usada para testar a plataforma.
